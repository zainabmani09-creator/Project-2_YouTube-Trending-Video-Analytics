📊 YouTube Trending Video Analytics
📌 Project Overview

This project analyzes YouTube trending video datasets across multiple regions to uncover patterns in content performance. The analysis focuses on category popularity, regional engagement, sentiment impact, and time-series trends.

🎯 Objectives
*Clean and standardize multi-region datasets
*Perform sentiment analysis on video titles
*Rank categories using SQL queries
*Build an interactive Tableau dashboard
*Generate business insights from trending data

🛠 Tools & Technologies
*Python (Pandas, Matplotlib)
*TextBlob (Sentiment Analysis)
*SQL
*Tableau Public
*Google Colab

📊 Dashboard Preview
Interactive Tableau Dashboard:
👉https://public.tableau.com/app/profile/zainab.mani/viz/Youtube_Trending_Video_Analytics/Dashboard1?publish=yes

📈 Key Insights
*Music is the highest-performing category.
*UK shows the highest average trending views.
*Negative sentiment titles tend to receive higher engagement.
*Trending performance shows seasonal variation across months.

📂 Project Structure
*data/ – Raw datasets
*notebook/ – Data cleaning and analysis notebook
*dashboard/ – Tableau dashboard link
*report/ – Final project report

🚀 Conclusion

This project demonstrates end-to-end data analytics workflow including preprocessing, sentiment analysis, SQL aggregation, visualization, and data storytelling.
